API Usage

curl -s "https://pos.rokomribazarshop.com/api/subdomain/?enum=YOUR_DOMAIN&key=YOUR_API_KEY" | jq -r '.subdomains[]'

Replace YOUR_DOMAIN with the target domain you want to scan for subdomains.

Parameters

  • enumTarget domain to enumerate
  • keyYour API authentication key

Response Format

{
    "domain": "hackerone.com",
    "total_found": 1504,
    "unique_subdomains_count": 92,
    "subdomains": [
        ".hackerone.com",
        "118.hackerone.com",
        "119.hackerone.com",
        "18.hackerone.com",
        ...
    ]
}
